# NDIV (Nuxt 3 + DaisyUI + Icones + VueUse) Starter

## Docs:
* [Nuxt 3](https://nuxt.com/docs/getting-started/introduction)
* [DaisyUI](https://daisyui.com/components/)
* [Icones](https://icones.js.org/)
* [VueUse](https://vueuse.org/functions.html)

## Extras
* [useAppHead](composables/useAppHead.ts) adds a title template
* [Vue 3.3](https://blog.vuejs.org/posts/vue-3-3#experimental-features) experimental features are on
## About
* My opinionated Nuxt starter kit
## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
